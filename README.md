# Волшебный стаканчик

Статическая обучающая веб-игра про состав чисел от 3 до 10.

## Запуск локально

Откройте `index.html` в браузере или запустите простой локальный сервер:

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

После этого игра будет доступна по адресу:

```text
http://127.0.0.1:4173/
```

## Размещение

Проект не требует сервера, базы данных и сборки. Его можно разместить на GitHub Pages:

1. Загрузить файлы проекта в репозиторий GitHub.
2. Открыть настройки репозитория.
3. Включить GitHub Pages для ветки с файлами.
4. Использовать ссылку GitHub Pages или подключить свой домен.

## Состав проекта

- `index.html` - разметка игры.
- `styles.css` - визуальный стиль и адаптивная верстка.
- `script.js` - логика заданий, звездочек, карты и наклеек.
- `ТЗ Волшебный стаканчик - доработанное.md` - актуальное техническое задание.
